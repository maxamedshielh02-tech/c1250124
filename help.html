How to Use This Website

1. Home Page

Home page-ka waxaad ka heli doontaa overview company-ga, services, team iyo button kuu diraya booking form.

2. Gallery Page

Gallery-ga waxaad ku arki doontaa projects (images) iyo table muujinaya project data sida client, status iyo year.

3. Booking Page

Booking page waa form aad ku dalban karto adeeg (Web Design, SEO, Branding). Buuxi form-ka kadibna submit garee.

4. About Page

About page wuxuu sharxayaa website-ka iyo developer-ka (adiga) iyo role-kaaga project-kan.

5. Navigation

Isticmaal navigation-ka kore si aad si fudud ugu dhex wareegto bogagga website-ka.

6. Images & Tables

Images waa portfolio, tables-na waa project data. Hover effects ayaa lagu daray si user experience u fiicnaato.
